# AI Sports Injury Risk Detection Platform

Developed as part of the **Infosys Virtual Internship**, this project provides a secure and scalable platform for athlete profile management and authentication, serving as the foundation for AI-based sports injury risk prediction.

## Milestone 1

### Completed Modules
- Backend setup using FastAPI
- Frontend setup using React and Vite
- PostgreSQL database integration
- User authentication (Registration & Login with JWT)
- Athlete Profile Management (Create, Read, Update and Delete)
- Backend and Frontend API integration
- API validation using Swagger UI
- Version control with Git and GitHub

### Technology Stack
- React
- Vite
- FastAPI
- Python
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Passlib (bcrypt)
- Git & GitHub
